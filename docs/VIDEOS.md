## Public Media Boundary

Videos and screenshots should not show:

- Real server IP addresses unless deliberately public.
- Root passwords, SSH keys, API keys, database passwords, or license keys.
- Customer data, customer domains without permission, invoices, emails, or private support tickets.
- Internal admin-only paths or hidden implementation details that weaken security.
