## Why The Bundle Is Not Fully Mirrored Here

The GitHub repository should stay clean and useful for installers, documentation, issues, and Free Core collaboration. The official website remains responsible for:

- Release bundle hosting
- Pro licensing
- Commercial support
- Update signing and release staging
- Paid vendor features
- Billing and vendor-controlled services
