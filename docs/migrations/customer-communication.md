# Customer Communication Template

Use this as a safe, plain-English customer update when a hosting provider is replacing an old control panel with FreeWebPanel.

## Short Maintenance Notice

Subject: Hosting panel maintenance window

We are upgrading the hosting control panel that manages websites, SSL, files, email, databases, backups, and support tools on this server.

The server IP and hosted websites are planned to remain in place. During the window, panel access may be unavailable while websites continue to be checked and protected. If any website, email, or database service needs a restart, we will keep it as short as possible.

Maintenance window:

```text
Date:
Start:
Expected finish:
Emergency contact:
```

## After-Upgrade Notice

Subject: Hosting panel upgrade complete

The hosting panel upgrade is complete. Website, SSL, mail, SQL, and file access checks have been completed.

New panel access:

```text
Admin/customer URL:
Support contact:
```

If you see anything unusual, contact support with the domain name and a short description of the issue.

## What Not To Tell Customers

- Root passwords.
- Private server paths.
- Firewall rules.
- Internal takeover scripts.
- Customer data belonging to other accounts.
- Panic language.

Keep the message calm, short, and focused on what customers can use.
